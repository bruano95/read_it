  ## DESCRIPTION
  ### This is an easy to use book search engine. The user is able to search for books they would want to read. This is using MERN stack to provide a smooth operating system and ultimately a user friendly application.

  ## INSTALLATION
  ### To install this product the user must obtain the deployed link. The user can also clone the repo to use it on their own device.

  ## USAGE
  ### To use this application the user will first go to the deployed link and follow the instructions on how to search for a book they desire. The user can search based on a number of categories.
